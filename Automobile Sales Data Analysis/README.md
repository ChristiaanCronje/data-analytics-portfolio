This project explores a dataset of automobiles with details such as price, make, body style, engine size, fuel efficiency, and other attributes. The analysis uncovers key trends and relationships to understand factors driving vehicle pricing and efficiency.

<h1></h1>

<img src="https://github.com/ChristiaanCronje/data-analytics-portfolio/blob/main/IGNORE/Headers/Dataset.svg" alt="Dataset" height="25px"/>

&#8287;&#8287;• `Automobile_data.csv` - contains automobile specifications and pricing details.

<h1></h1>

<img src="https://github.com/ChristiaanCronje/data-analytics-portfolio/blob/main/IGNORE/Headers/Features.svg" alt="Features" height="25px"/>

&#8287;**Data Cleaning & Preprocessing**  
&#8287;&#8287;&#8287;&#8287;• Handling missing values and duplicates.  

&#8287;**Exploratory Data Analysis (EDA)**  
&#8287;&#8287;&#8287;&#8287;• Vehicle make distribution  
&#8287;&#8287;&#8287;&#8287;• Engine size vs. highway MPG  
&#8287;&#8287;&#8287;&#8287;• Body style distribution  
&#8287;&#8287;&#8287;&#8287;• Fuel type distribution  
&#8287;&#8287;&#8287;&#8287;• Price analysis (most vs. least expensive)  

&#8287;**Visualizations & Insights**  
&#8287;&#8287;&#8287;&#8287;• Clear visuals and insights for non-technical stakeholders.

<h1></h1>

<img src="https://github.com/ChristiaanCronje/data-analytics-portfolio/blob/main/IGNORE/Headers/How%20to%20Run.svg" alt="How to Run" height="25px"/>

1. Open `Automobile_EDA.ipynb` in Jupyter Notebook or VS Code with Python support.
2. Run each cell sequentially to reproduce analyses and visualizations.
3. Review insights in the markdown cells accompanying the charts.
