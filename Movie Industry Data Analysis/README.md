This project analyzes a dataset of movies, including details such as budget, revenue, popularity, vote average, and genre. The analysis identifies trends and patterns that help understand factors influencing a movie’s performance.

<h1></h1>

<img src="https://github.com/ChristiaanCronje/data-analytics-portfolio/blob/main/IGNORE/Headers/Dataset.svg" alt="Dataset" height="25px"/>

&#8287;&#8287;• `Movies_data.csv` — contains movie information such as budget, revenue, popularity, votes, and genres.

<h1></h1>

<img src="https://github.com/ChristiaanCronje/data-analytics-portfolio/blob/main/IGNORE/Headers/Features.svg" alt="Features" height="25px"/>

&#8287;&#8287;**Data cleaning and preprocessing, including handling missing values and duplicate rows.**  

&#8287;&#8287;**Exploratory Data Analysis to uncover insights:**  
&#8287;&#8287;&#8287;&#8287;• Frequency of movies by genre  
&#8287;&#8287;&#8287;&#8287;• Budget vs. revenue relationship  
&#8287;&#8287;&#8287;&#8287;• Movie production trends by decade  
&#8287;&#8287;&#8287;&#8287;• Distribution of movie ratings and popularity  
&#8287;&#8287;&#8287;&#8287;• Profit vs. budget analysis  

&#8287;&#8287;**Visualizations and summaries suitable for clients and stakeholders.** 

<h1></h1>

<img src="https://github.com/ChristiaanCronje/data-analytics-portfolio/blob/main/IGNORE/Headers/How%20to%20Run.svg" alt="How to Run" height="25px"/>

1. Open `Movies_EDA.ipynb` in Jupyter Notebook or VS Code.
2. Execute cells sequentially to reproduce analysis and visualizations.
3. Insights are presented in markdown cells throughout the notebook.
