<img src="https://github.com/ChristiaanCronje/data-analytics-portfolio/blob/main/IGNORE/Headers/Description.svg" alt="Description" height="30px"/>

This project explores the `penguins_size` dataset, which contains physical measurements of penguins, including culmen length and depth, flipper length, body mass, species, sex, and island inhabited. The analysis examines relationships between physical features and differences across species and sexes.

<h1></h1>

&#8287;&#8287;• `penguins_size.csv` — contains measurements of penguins across different species and islands.
